# Academic Achievement & Analytics Management System

An enterprise-level full-stack web application designed for comprehensive college management.

## Features
- **Role-Based Access**: Student, Faculty, HOD, Principal, Admin.
- **Dashboards**: Customized views for each role.
- **Analytics**: Graphic visualization of department performance.
- **Reports**: Auto-generated PDF reports.
- **Security**: JWT, BCrypt, Helmet, XSS-Clean.

## Setup
### Backend
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Deployment
- Backend: Render/Railway
- Frontend: Vercel/Netlify
